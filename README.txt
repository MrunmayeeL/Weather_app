Overview

This app allows you to check the current weather for any city. It shows the temperature, weather description, humidity, wind speed, and an icon based on the weather condition.
Features

    Search weather by city.
    Displays temperature, description, humidity, and wind speed.
    Shows a weather icon (e.g., sunny, rainy, snowy).

Requirements

    OpenWeatherMap API: You need an API key from OpenWeatherMap.

How to Use

    Get an API Key from OpenWeatherMap.
    Replace the api_key in the script with your key.
    Open the HTML file in a browser.
    Enter a city and click Search to see the weather.
